# RB

*Finally! [**Released**](https://github.com/gomorizsolt/react-typescript-boilerplate) the boilerplate with TypeScript support.*

## Core features
- ESlint
- Jest & Enzyme
- Pre-commit
- Prettier
- CSS loader & styled-components
- Environment variables
- Automated Heroku deployment flow

## How to use?

Start off by cloning the project.

```bash
git clone https://github.com/gomorizsolt/react-boilerplate.git
```

Install dependencies.

```bash
npm install
```

Launch the development environment. :rocket:

```bash
npm run dev
```
